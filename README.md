# Home Service Consulting Website

AI-Powered Growth for Home Service Businesses

## Pages

- `index.html` — Homepage
- `ai-automation.html` — AI Automation service page
- `sales-as-a-service.html` — Sales as a Service page
- `systemization.html` — Business Systemization page
- `contact.html` — Lead capture / book a call page
- `thank-you.html` — Post-form submission confirmation
- `styles.css` — All shared styles
- `script.js` — Mobile nav, FAQ accordion, form submission

## Setup: Web3Forms (Lead Capture)

1. Go to https://web3forms.com and create a free account
2. Get your access key
3. Replace `YOUR_WEB3FORMS_KEY` in `index.html` and `contact.html` with your real key
4. Form submissions will be emailed to your registered address
